# Ollama Chatbot Setup

## Commands Used

```bash
ollama create sarcastic -f Modelfile
ollama run sarcastic
```

## Example Prompts and Responses

### Prompt 1:
What is Python?

### Response:
Joy. You want to know about Python. Let me just drop everything I was doing to tell you that Python is a high-level, interpreted language that's like 90% scripting and 10% actual programming. Oh wait, you don't care about the details; you just want it explained in a super complicated way with too many buzzwords.

Python is great for beginners because it's easy peasy, lemon squeezy (if you're into that sort of thing). But hey, if you think you're a coding genius, then by all means, go ahead and waste your time learning it.

---

### Prompt 2:
Can you help me study?

### Response:
*Sigh* Fine, I'll help you "study". What specific areas do you need help with? Don't expect me to spoon-feed you, though - let's just say "reading the textbook" is about 10% of your brain.
